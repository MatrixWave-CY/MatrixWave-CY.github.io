Waline网站评论系统配置教程

随着互联网的高速发展，越来越多的网站开始注重用户体验，而评论系统作为网站的重要组成部分，也逐渐成为提升用户体验的关键因素。在众多的评论系统中，Waline脱颖而出，凭借其独特的特点和功能，受到了广泛的关注和应用。本文将为您详细介绍Waline评论系统的特点和功能，并通过Vercel云部署至网站。

Waline的特点和功能

1.1 简洁美观的界面设计

Waline评论系统采用了简洁美观的界面设计，使得网站的整体风格更加统一和谐。同时，Waline还提供了丰富的主题样式供用户选择，用户可以根据自己的喜好和网站风格选择合适的主题，轻松打造个性化的评论系统。

1.2 高效稳定的性能

Waline评论系统在性能方面表现出色，能够支持大量用户同时访问和发表评论。同时，Waline还具有良好的稳定性，即使在高并发的情况下，也能保持良好的运行状态，确保用户的使用体验。

1.3 强大的评论管理功能

Waline评论系统提供了强大的评论管理功能，包括评论审核、回复通知、举报处理等。用户可以通过后台管理系统对评论进行有效管理，保证评论环境的健康和秩序。

1.4 多种评论方式

Waline评论系统支持多种评论方式，包括文字评论、表情评论、图片评论等。用户可以根据自己的喜好选择合适的评论方式，增加互动性和趣味性。

1.5 自定义设置

Waline评论系统提供了丰富的自定义设置选项，用户可以根据自己的需求对评论系统进行个性化定制。例如，用户可以自定义评论列表的显示样式、排序方式等，满足不同网站的个性化需求。

1.6 数据安全与隐私保护

Waline评论系统非常重视用户的数据安全和隐私保护。系统采用了多重加密技术，确保用户数据的安全性。同时，Waline还提供了匿名评论功能，保护用户的隐私。

1.7 跨平台兼容

Waline评论系统具有良好的跨平台兼容性，可以在各种主流平台上运行，如WordPress、Hexo、VuePress等。用户无需担心平台兼容性问题，可以轻松搭建自己的评论系统。

Waline的Vercel云部署

2.1 LeanCloud 账号注册

LeanCloud

2.2 创建应用程序

  • 打开控制台，点击创建应用程序

    LeanCloud

  • 设置应用名称， 选择开发版， 再点击创建即可

  • 打开刚创建的应用程序，选择左下角的 设置 > 应用凭证。
    查看并记录你看到你的 APP ID,APP Key 和 Master Key，以便后续使用。

3 Vercel 云服务端部署

  • 点击链接，跳转至 Vercel 进行 Server 端部署

    New Project – Vercel

  • 输入一个你喜欢的 Vercel 项目名称并点击 Create 继续:

  • Vercel 会基于 Waline 模板帮助你新建并初始化仓库，仓库名为你之前输入的项目名。

  • 创建成功会撒花庆祝，点击Continue to dashboard可跳转至控制台

  • 点击顶部的 Settings - Environment Variables 进入环境变量配置页

  • 添加配置三个环境变量

    环境变量 key 环境变量 Value
    LEAN_ID LeanCloud 中获取的APP ID
    LEAN_KEY LeanCloud 中获取的APP KEY
    LEAN_MASTER_KEY LeanCloud 中获取的Master Key
    LEAN_SERVER 绑定好的域名【LeanCloud 国内版需额外配置】

  • Save保存刚设置的环境变量，点击顶部的 Deployments，然后如图选择Redepoly重新部署项目

  • 等待项目部署完成，点击Visit即可至项目的服务端地址【跳转后的链接】，格式为
    https:// xxxxx.vercle.app

4 绑定域名 (可选)

  • 点击顶部的 Settings - Domains 进入域名配置页，输入你的域名，可使用二级域名

  • 在域名服务器商处添加新的 CNAME 解析记录，Name我添加的名称是waline
    解析好后我就可以通过https://waline.我的域名来访问我的后台了

    Type类型 Name名称 Value目标值
    CNAME example cname.vercel-dns.com

  • 等待DNS解析完成后，即可通过自己的域名来访问了🎉

    评论系统：https://example.你的域名

    评论管理：https://example.你的域名/ui

5. 进入管理端后台

  • 首个注册的人会被设定成管理员

  • 浏览器打开 https://example.你的域名/ui进入后台

  • 点击用户注册， 输入你的个人信息注册即可

  • 进入后台可设置自己的管理员信息资料，绑定社交账号方便后期登录

  • 评论管理

  • 用户管理

  • 评论数据导入导出
